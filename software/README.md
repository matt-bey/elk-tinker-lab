# Software

OS configuration and software setup topics that apply across hardware combinations.

## Subdirectories

- [retropie/](retropie/) — RetroPie setup, display and audio configuration
- [raspberry-pi-os/](raspberry-pi-os/) — Raspberry Pi OS headless setup, SSH, WiFi

## What Goes Here vs Docs

`software/` covers how to configure specific software packages and operating systems. `docs/` covers hardware-level topics (GPIO numbering, SPI vs I2C, power budgeting) that span everything.
